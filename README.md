# YOLOv8 Object Detection System

## Overview

This project is a YOLOv8-based object detection system trained on the COCO 2017 dataset. It is designed to detect and classify multiple objects across different real-world environments including traffic scenes, homes, wildlife areas, and classrooms.

The system demonstrates practical applications of deep learning in computer vision.

## Features

- YOLOv8 deep learning model for object detection
- Trained on COCO 2017 dataset
- Detects 30+ object categories
- Multi-environment detection:
  - Traffic scenes
  - Home environments
  - Wildlife detection
  - Classroom settings
- Real-time object detection capability
- Bounding box visualization

## Tech Stack

- Python
- YOLOv8 (Ultralytics)
- OpenCV
- PyTorch
- COCO 2017 Dataset

## Object Categories

The system can detect objects such as:
- People
- Vehicles (cars, buses, trucks)
- Animals
- Furniture
- Classroom objects
- Everyday household items

## System Workflow

1. Input image or video is provided
2. YOLOv8 model processes the input
3. Objects are detected and classified
4. Bounding boxes are drawn on detected objects
5. Results are displayed in real-time

- Smart surveillance
- Wildlife observation
- Educational AI demonstrations
- Home security systems
